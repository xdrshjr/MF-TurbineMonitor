## 项目说明

目前项目有多个分支版本:

- `v1.0.0` 分支是基于`vue2+webpack`架构搭建的，实现大部分基本功能，但代码没有做
  抽象和梳理, 阅读体验可能较差。
- `v2.0.0`以上版本使用`vue3+ts+vite`对项目进行重构，在功能和样式上与`v1.0.0`有些
  许差异。
